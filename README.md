# SpamGuard-AI

Projeto de Inteligência Artificial para classificação de mensagens como spam ou não spam,
utilizando processamento de linguagem natural (NLP) e Machine Learning.

## Tecnologias utilizadas
- Python
- scikit-learn
- Naive Bayes
- CountVectorizer

## Funcionalidades
- Treinamento de modelo simples de classificação
- Análise de texto digitado pelo usuário
- Retorno indicando se a mensagem é spam ou não

## Como executar
```bash
pip install scikit-learn
python main.py
